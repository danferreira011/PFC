O projeto esta criado na linguagem PHP OO, utilizado como SGBD o mySQL, então para testá-lo é necessário ter os mesmos instalados na maquina. Particularmente utilizo o programa WAMP (que pode ser baixado livremente da WEB), que instala automaticamente tudo que você precisa para rodar aplicações em PHP.


Descompacte o arquivo PFC.rar na pasta de trabalho do servidor, que no caso do wamp é <www>. Recuperar o banco dbpfc.sql no mysql e configurar o arquivo conexaobr.inc com os dados do seu SGBD.

Apos, tudo realizado, para testar o programa abra o navegador e digite: http://localhost/ então você terá acesso á pasta no navegador. Então basta selecionar o arquivo na pasta Interfaces_PFC para testar os mesmos.
